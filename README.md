# Documentacion
1. Ejecutar docker en tu escritorio
2. Utilizar cd "(carpeta de docker compose)
3. Ejecutar docker compose up -d en la terminal"
4. El archivo docker compose leera automaticamente el respaldo y reparara la base de datos
5. Si la base de datos no se crea correctamente ejecutar docker compose down -v y probar de nuevo
6. Abre el archivo app folder
7. Usa cd para abrir la carpeta de app en la terminal cd "\taller 2\app_folder"
8. Ejecutrar pip install -r requirements.txt
9. Ejecuta app.py
